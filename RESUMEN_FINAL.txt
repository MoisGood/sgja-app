TU PREGUNTA:
└─ "¿Cómo sabe el sistema si el usuario cerró el navegador sin hacer logout?"

RESPUESTA:
└─ Por HEARTBEAT (latido) - se envía cada 30 segundos
   Si para de llegar → Sesión muerta → Se cierra

🔄 FLUJO DE LOS 3 CASOS
═══════════════════════════════════════════════════════════════════════════════

CASO 1: LOGOUT NORMAL
  Usuario → Click "Cerrar Sesión" → registrarCierre() → ✅ Cerrado
  Costo: $0

CASO 2: INACTIVIDAD 30 MINUTOS
  Usuario → Abre app pero no hace nada 30 min
         → En Seguridad tab: "Inactivo 30+ min"
         → Click "Cerrar inactivas" → ✅ Cerrado
  Costo: $0

CASO 3: NAVEGADOR CERRADO (TU PROBLEMA)
  Usuario → Cierra navegador sin logout
         → ❌ Heartbeat DEJA de llegar
         → Sistema espera 5 minutos
         → ¿Heartbeat? No
         → Opción A: Admin lo ve en Seguridad → Click cerrar ✅ ($0)
         → Opción B: Cloud Function lo cierra automático ✅ ($2-5/mes)
